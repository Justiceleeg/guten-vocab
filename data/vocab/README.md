# Vocabulary Lists

This directory contains grade-level vocabulary word lists for grades 5-8.

## Source

Vocabulary words are sourced from [Vocabulary.com Lists by Grade](https://www.vocabulary.com/lists/lists-by-grade).

## Files

- `5th_grade.json` - 150 words
- `6th_grade.json` - 125 words
- `7th_grade.json` - 125 words
- `8th_grade.json` - 125 words

**Total: ~525 unique words** (some words may appear in multiple grade levels)

## Usage

These files are used by `scripts/seed_vocabulary.py` to populate the `vocabulary_words` table in the database.
